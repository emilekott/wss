Method Management

The first screen to be seen upon going to Modules > Advanced Shipper Config is the Method Management utility.

This utility shows the ordered list of the shipping methods defined for the current configuration.

Each method is numbered and may have an optional Method Admin Title displayed - a quick and intuitive way of telling one method from another within the admin - this is not shown to the customer!

Adding/Configuring A Method

An existing method can be configured by clicking its Edit button, whereas a new method can be added by clicking the Insert button at the bottom of the page (if present - it is only displayed on the last page).

Method Summaries/Controls

Clicking on “Show Method Summary & Other Controls” shows an overview of the main settings for a method and reveals some controls for inserting, copying and deleting the method.

Copy Method

The purpose of the Copy Method control is to quickly and easily base the configuration of a new method on another, pre-configured method.

For Example: If an “Overnight Ground” service has different prices for sending a Dining Table over a Bookcase, and the calculations can't be done automatically by the weight of these products (as entered in the admin), then:

  • once an “Overnight Ground” method has been set up for the Dining Table for all the different regions it can be sent to,
  • this method can be copied as a new method,
  • and the product selection and table of rates modified for the Bookcase.

Altogether much quicker than setting everything up again from scratch for the Bookcase!
