Quickstart Examples

The following examples are intended to show how various common shipping requirements for a store may be set up using the Advanced Shipper module.

It is advised that you read the full documentation as all the information you could possibly need is there (and a huge amount of effort has been put in to make it logical, easy to follow and easy to dip in and out of)!

We know that some people just don't like reading docs though, so hopefully these examples help...

Example 1 - £4.50 for the consignment and then £0.50 per Kg.

There are two ways this can be accomplished:

  1. A Flat & Percentage Rate:

    Table of Rates: <weight>*:4.50+50%</weight>

  2. A Percentage Rate and a Surcharge:

    Table of Rates: <weight>*:50%</weight>
    Surcharge: 4.50

Example 2 - £8.50 for the first 15Kg and then £0.27 per Kg thereafter.

This can be accomplished using Combination Rates:

Table of Rates: <weight>*:(0-15:8.50)(*:27%)</weight>

Example 3 - £0.40 per Kg but only up to £30 worth of applicable products.

This can be accomplished using Nested Tables of Rates:

Table of Rates: <price>0-30:<weight>*:40%</weight></price>

Example 4 - If the order is under $200 and it's national (US), it uses a UPS calculator. If the order is over $200 and it's national (US), it will be free shipping. If the order is international (non-US), it uses a USPS calculator (regardless of the order total).

This can be easily set up using the following configuration:

  1. Add a method, with the method title “UPS Shipping”.

  2. Add a region for this method:

    Countries/Postcodes: US
    Table of Rates: <totalorderprice>0-200:ups, *:0</totalorderprice>

  3. Add another method, with the method title “International Shipping”.

  4. Add a region for this method:

    Countries/Postcodes: !US, *
    Table of Rates: <totalorderprice>*:usps</totalorderprice>

Example 5 - Applicable products total between £0-£25: £2.50 per Kg, £25-£100: £1.50 per Kg

This can be accomplished using Nested Tables of Rates:

Table of Rates: <price>0-25:<weight>*:250%</weight>, 25-100:<weight>*:150%</weight></price>
