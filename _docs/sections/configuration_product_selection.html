Product Selection

There are two main ways in which products can be covered by a method.

  1. Method as a “Fallover”.

    If a method is set up to act as a “Fallover”, then it will apply for all products which don't otherwise have a method defined.

    For Example: If a store has 26 products, named ‘A’-‘Z’ and the product named ‘C’ has been selected as being covered by another shipping method, but no other products have been associated with another shipping method, then this method will be applicable for the products named ‘A’, ‘B’ and ‘D’-‘Z’.

    Even though a method can be set to be acting as a Fallover, it can still have individual products and/or categories and/or manufacturers specified as being applicable for it (covered by it).

    The reason for this is to allow individual products/categories/manufacturers to have specific shipping method(s) set up for them but still be included by this method.

    For Example: In the case of an online comic retail store which has a general shipping method that applies to all products on the site (as it has been configured as a Fallover method):

    If a 1st edition of a particular comic could be shipped either using the store's “Standard” shipping method, or via a special “Signed For” Courier Service, the Advanced Shipper module wouldn't see the store's “standard” method (the Fallover) as applying to/covering the 1st edition comic.

    This would be because the module would see that this 1st edition comic already has a specific method set up for it, which would be determined to be the exclusive method for that comic.

    Specifically selecting this 1st edition comic for the “Standard” method as well means that both methods can cover the shipping of the comic and could possibly both be presented as a shipping method option upon checkout.

  2. Method applies for specific categories and/or manufacturers and/or products only.

    If a method is set up to apply for specific categories and/or manufacturers and/or products only then it can only be an option for the categories and/or manufacturers and/or products specified.

    By selecting a product/category/manufacturer as having this method apply to it, that product/category will only be able to be shipped by this method, unless the product/category/manufacturer is also specifically selected for other method(s) as well.

Selecting a Category/Categories of Products

Selecting a category/categories of products as being applicable for a method is easily done.

  • Simply use Insert to bring up the “Categories Selection” Popup and select the desired category/categories from the list.

Selecting Manufacturer(s) of Products

Selecting a manufacturer(s) of products as being applicable for a method is easily done.

  • Simply use Insert to bring up the “Manufacturer(s) Selection” Popup and select the desired manufacturer(s) from the list.

Selecting a Product

Selecting a specific product as being applicable for a method is also a simple matter.

  • Upon using Insert to bring up the “Product Selection” Popup, a list of the store's categories is displayed.

  • Once a category containing products has been selected, a product from that category can be selected.

    • If the product has any options, a specific set of options can be selected, so that the shipping method will apply only if a product with this specific set of options/attributes is in the customer's cart. Alternatively, “Apply For All Product Options” can be selected, so that any set of options/attributes for this product will be covered by this shipping method.

    • If a product has no options/attributes then no further selections are required.
