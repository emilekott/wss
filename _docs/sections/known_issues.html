Known Issues

Although months of work have gone into bringing the module to the capable stage it is at, there are still some aspects of the module which could be improved, so it is best to be aware of the limitations of the module as it stands...

  • Take Care when Configuring the Shipping Methods and Regions

    The configuration utility does not yet carry out full error checking of values entered. A certain level of error checking, validation and notification takes place, but it is essentially up to you to make sure the configuration settings make sense! This is a very flexible module so preventing illogical configuration setting combinations is impossible; please take care when configuring it!

    Better, context-sensitive error reporting may be added in the future, which should aid identifying configuration problems.
